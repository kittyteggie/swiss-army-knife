# Swiss Army Knife: Multiple mini-programs in one
Swiss Army Knife (or SAK) is a simple program with multiple useful (unuseful) tools written in C, it is currently a HUGE work in progress because I have plans for adding other tools.  

# Compiling
For compiling you just need to get the `main.c` file and use gcc, a command like this:  
`gcc main.c -o knife` (I personally use `knife` as a name but you can use `sak` or anyone you like)  

Then run it: `./knife`.  
  
# Current tools
* `btd` Binary to Decimal converter
* `calc` Calculator
* `dtb` Decimal to Binary converter  
* `unixt` Current Unix time
* `unixtc` Unix time
